# Ejercicios de condicionales en Java.
> Diferentes ejercicios para practicar los condicionales en Java, algunos de ellos propios de las clases de DAW.

1. Realiza un programa que reciba dos números por teclado e indique **cuál es el mayor y si son iguales.**
   
2. Realiza un programa que pida un número por teclado y nos indique si es **par o impar.**
   
3. Crear un programa que pida al usuario dos números y:
    - Muestre el resultado de su división.
    - Pero... si el segundo número es 0, muestre un mensaje de error y finalice el programa.

4. Realizar un programa que lea una cadena de caracteres (String) por teclado y compruebe si es una letra mayúscula.
